SBOLObject Class Reference

Detailed Description

An SBOLObject converts a C++ class data structure into an RDF triple store and contains methods for serializing and parsing RDF triples.

 The identity property is REQUIRED by all Identified objects and has a data type of URI. A given Identified objectâ€™s identity URI MUST be globally unique among all other identity URIs. The identity of a compliant SBOL object MUST begin with a URI prefix that maps to a domain over which the user has control. Namely, the user can guarantee uniqueness of identities within this domain. For other best practices regarding URIs see Section 11.2 of the SBOL specification doucment.

void close ( )
virtual

Use this method to destroy an SBOL object that is not contained by a parent Document.

If the object does have a parent Document, instead use doc.close() with the object's URI identity as an argument. Recurse through child objects and delete them.
